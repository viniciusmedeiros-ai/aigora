O Dashlane disse que os invasores montaram uma campanha coordenada de hackers contra uma grande base de seus usuários, na tentativa de recuperar o maior número possível de cofres de senhas criptografadas. O provedor do gerenciador de senhas disse que menos de 20 cofres pessoais de usuários foram baixados antes de encerrar a operação.

Em uma campanha que começou no domingo, o ator de ameaças desconhecido abusou do mecanismo que permite que os usuários do Dashlane adicionem novos dispositivos, como computadores ou telefones, às suas contas. Ao abusar das interfaces de programação do Dashlane para o registro de dispositivos, os invasores enviaram solicitações para um grande número de endereços de e-mail registrados dos usuários existentes. Em uma atualização publicada na quinta-feira, o Dashlane escreveu:

O agente da ameaça direcionou os endpoints da API para o registro do dispositivo e usou um ataque de força bruta para enviar um grande volume de solicitações automatizadas para esses endpoints.

Em resposta, os sistemas de segurança automatizados da Dashlane funcionaram como pretendido, desencadeando um bloqueio automático das contas visadas para proteger esses utilizadores. Antes que o ataque fosse totalmente mitigado, o agente de ameaças foi capaz de usar força bruta e gerar tokens válidos para menos de 20 clientes de planos pessoais, permitindo que eles registrassem um novo dispositivo nessas contas e baixassem cópias das contas criptografadas dos usuários Cofres
