A Microsoft acaba de lançar o Build 2026 com uma palestra do CEO Satya Nadella e outros líderes da empresa. Como esperado, ele estava repleto de anúncios, desde o novo hardware Surface até um assistente pessoal sempre ativo e atualizações nos modelos internos de IA da Microsoft.

Se você não assistiu ao evento ao vivo, pode acompanhar todas as últimas notícias no resumo abaixo.

O Surface RTX Spark Dev Box é voltado para desenvolvedores que desejam executar modelos locais de IA em seus dispositivos, servindo como um substituto para o kit de desenvolvimento cancelado da Qualcomm. Ele vem equipado com o novo chip Spark RTX baseado em ARM da Nvidia e 128 GB de memória unificada, além de aplicativos pré-instalados, como o Visual Studio Code e o GitHub Copilot. O Surface RTX Spark Dev Box também tem uma versão pré-configurada do Windows 11 Pro com modo escuro ativado por padrão, uma barra de tarefas simplificada e sem widgets.

A Microsoft ainda não revelou os preços ou a lista completa de especificações, mas o dispositivo estará disponível nos EUA ainda este ano.

A Microsoft está tomando medidas para tornar o Windows mais amigável para desenvolvedores. Isso inclui a adição do Coreutils, que a empresa descreve como "utilitários de linha de comando semelhantes ao Linux que são executados nativamente" no Windows 11. Também está lançando a capacidade de criar, executar e interagir com contêineres Linux por meio de seu Subsistema Windows para Linux (WSL), juntamente com um novo Terminal Inteligente que fornece contexto para o agente alimentado por IA preferido de um desenvolvedor.

Tivemos um vislumbre do Project Solara, um sistema operacional baseado em Android projetado para executar agentes em uma variedade de dispositivos. A Microsoft fez uma parceria com a Qualcomm e a MediaTek para desenvolver o sistema, que poderia eventualmente funcionar como um complemento para um PC ou transferir tarefas entre dispositivos. Durante a palestra, a Microsoft mostrou alguns dispositivos de exemplo em que a tecnologia pode ser executada: um hub de desktop e um crachá digital.

A Microsoft está lançando um assistente sempre ativo baseado no OpenClaw, a plataforma de IA de código aberto que ganhou popularidade no início deste ano. O assistente, chamado Scout, trabalha com aplicativos do Microsoft 365, incluindo Outlook, OneDrive e Microsoft Teams. Ele foi projetado para executar tarefas em segundo plano, ajudando as empresas a organizar calendários, gerenciar relatórios de despesas, escrever e-mails e muito mais.

O Scout faz parte de um conjunto mais amplo de agentes de "piloto automático" que a Microsoft planeja lançar, cada um dos quais terá sua própria "identidade".
