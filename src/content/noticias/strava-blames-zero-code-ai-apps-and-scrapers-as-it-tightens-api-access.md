A popular plataforma de monitoramento de condicionamento físico, Strava, está restringindo o acesso à sua API como parte dos esforços para reprimir a raspagem de IA, conforme relatado anteriormente pelo TechCrunch . Os desenvolvedores que desejam criar um aplicativo usando os dados do Strava agora precisam pagar por uma assinatura fixa de $ 11,99 / mês.

Em uma atualização em seu hub de desenvolvedores, o Strava culpa a mudança em “ferramentas de IA de código zero” que permitem aos usuários criar rapidamente aplicativos que “martelam” APIs. "Sentimos isso em primeira mão — os aplicativos de desenvolvedores para o nosso programa aumentaram 448% no acumulado do ano, os intermediários da API violaram os termos da política e as tentativas de extração degradaram o desempenho da plataforma para todos", escreve a empresa. Da mesma forma, o Reddit começou a cobrar dos desenvolvedores para acessar sua API em 2023 .

Antes dessa mudança, os desenvolvedores podiam se inscrever para usar a API do Strava gratuitamente, com a capacidade de expandir seu acesso à medida que adicionassem mais usuários, relata o TechCrunch. Este não é o primeiro passo que o Strava deu para restringir o acesso à sua plataforma, já que a empresa começou a restringir os dados que aplicativos de terceiros podem mostrar em 2024 . Também processou a parceira de longa data Garmin por supostamente infringir suas patentes, mas depois desistiu do caso . A Strava entrou com pedido de oferta pública inicial em fevereiro .

Além disso, os usuários do Strava agora podem usar uma nova ferramenta que permite vincular seus dados de condicionamento físico — incluindo ritmo, frequência cardíaca por segundo, dados de GPS e muito mais — ao Claude. A empresa diz que a nova restrição de API não afetará as integrações de wearables e dispositivos ou a capacidade dos usuários de baixar seus dados gratuitamente.
