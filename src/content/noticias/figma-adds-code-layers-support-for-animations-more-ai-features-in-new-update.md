O Figma exibiu na quarta-feira uma atualização que adiciona uma nova camada de código, suporte para motion e shaders e a capacidade de criar plug-ins personalizados para várias tarefas usando IA.

A plataforma de design vem trabalhando há algum tempo para trazer a integração de código para sua ferramenta. No ano passado, revelou uma ferramenta de prototipagem baseada em prompt de IA, o Figma Make, e desde então lançou integrações com o Claude Code e o Codex para melhorar a transferência entre codificação e design.

A empresa agora está adicionando camadas de código diretamente à tela colaborativa, ajudando as equipes a clonar repositórios e extrair fluxos do código para projetar camadas para testes.

O diretor de produtos da Figma, Yuhki Yamashita, disse que as camadas de código tornam mais fácil para designers, gerentes de produto e programadores iterar ideias, em vez de se concentrar na criação de código puro que entra em produção.

“Achamos que a tela multiplayer é realmente poderosa porque este é um ambiente onde você realmente não se importa com a qualidade do código. Se você está explorando rapidamente ou precisa explorar um monte de novas direções, pode fazer isso dessa maneira espacial. Esperamos que esse recurso produza um comportamento diferente não apenas com designers, mas também com engenheiros e PMs”, disse ele durante uma ligação.

O Figma agora também suporta animações, transições e transformações 3D. Anteriormente, os designers precisavam criar animações em outros softwares e convertê-las em código que o aplicativo pudesse entender. Agora, os designers podem integrar animações e transições diretamente no Figma.

Agora você pode usar a IA para criar alguns desses ativos, e a atualização também está adicionando suporte para adicionar efeitos de sombreamento e preenchimentos usando IA.

No ano passado, o Figma adquiriu a ferramenta baseada em nós Weavy, que ajudou os designers a executar fluxos de trabalho por meio de diferentes modelos para comparar as saídas, e agora está trabalhando para integrar melhor os dois aplicativos. Em uma atualização lançada no final do ano, os usuários poderão gerar fluxos de trabalho pesados diretamente no Figma.
