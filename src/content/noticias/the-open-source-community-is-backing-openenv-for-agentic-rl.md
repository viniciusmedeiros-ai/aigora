O OpenEnv é uma ferramenta para criar um ambiente de execução agêntico, como terminais, navegadores ou qualquer coisa com a qual um agente possa interagir. E hoje, temos o prazer de anunciar que a OpenEnv está se tornando ainda mais aberta, para tornar o futuro dos agentes de treinamento de código aberto.

A partir de hoje, a OpenEnv será coordenada por um comitê que até agora inclui Meta-PyTorch, Reflection, Unsloth, Modal, Prime Intellect, Nvidia, Mercor, Fleet AI e Hugging Face. OpenEnv agora vive em huggingface/OpenEnv

O projeto OpenEnv é apoiado e adotado por algumas das principais organizações do ecossistema de IA, incluindo PyTorch Foundation, vLLM, SkyRL (UCB), Lightning AI, Axolotl AI, Stanford Scaling Intelligence Lab, Mithril, OpenMined, Scaler AI Labs, Scale AI, Patronus AI, Surge AI, Halluminate, Turing, Scorecard e Snorkel AI.

Arneses de agentes como Claude Code, Codex, OpenClaw e Hermes continuam melhorando. Uma razão para sua melhoria é que modelos como GPT-5.5 e Opus 4.8 são treinados para usar seus respectivos chicotes.

Também queremos esses ganhos com modelos de código aberto: treinar modelos locais que usem os chicotes de forma eficaz e economizar computação, especializando modelos para tarefas específicas.

Os laboratórios Frontier treinam modelos e arreios que, na maioria das vezes, funcionam como luvas. O modelo é treinado para usar o arnês e otimizado para suas características. Os modelos podem generalizar além desses arreios, até certo ponto, mas nada supera a eficiência do treinamento.

A céu aberto, esse não é o caso. Os desenvolvedores usam qualquer arnês, qualquer modelo, qualquer mecanismo de inferência, em qualquer caso de uso que valorizem. Isso é fundamental para a comunidade, mas também é um desafio que requer infraestrutura e ferramentas para enfrentar.

É aí que entra a OpenEnv. É uma biblioteca para fazer a interface entre arnês, ambiente e treinador, que funciona em qualquer modelo. Para que isso se mantenha, ele precisará ser propriedade de todas as principais partes interessadas.
