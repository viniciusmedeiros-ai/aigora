O Alibaba da China proibirá os funcionários de usar a ferramenta de programação Claude Code da Anthropic, a partir de 10 de julho, de acordo com vários relatórios .

A Anthropic já proíbe que empresas chinesas, bem como entidades estrangeiras de propriedade dessas empresas, usem seus modelos. A empresa vem trabalhando para fechar brechas que permitem que os usuários chineses acessem Claude.

De acordo com uma publicação recente no Reddit, parte desse fechamento de lacunas envolveu uma versão do Claude Code que poderia identificar secretamente usuários chineses. Thariq Shihipar, da Anthropic, disse em um post no X que este era "um experimento que lançamos em março que visava evitar o abuso de contas de revendedores não autorizados e proteger contra a destilação." ( A destilação é uma prática em que os modelos de IA são treinados nas saídas de outros modelos.)

"A equipe conseguiu mitigações mais fortes desde então e, na verdade, pretendemos derrubar isso por um tempo", disse Shihipar.

No entanto, o Alibaba classificou o Claude Code como um software de alto risco e está instruindo os funcionários a usar a própria ferramenta Qoder da empresa.
