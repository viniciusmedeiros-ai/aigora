A era tokenmaxxing do Vale do Silício agora tem seu próprio hardware. Um novo projeto de código aberto traz suas estatísticas de utilização do Claude Code para um pequeno painel de desktop, permitindo que os usuários avançados de IA fiquem de olho em seu uso.

Claro, você pode rastrear o uso do Código Claude diretamente no terminal usando comandos ou outras ferramentas e aplicativos externos, mas isso não é tão divertido quanto ver uma versão pixel-art da dança do sprite Clawd em uma tela antes de mostrar informações de uso de token de relance, não é?

O "Clawdmeter", como o dispositivo é chamado, é um projeto paralelo divertido para usuários avançados de IA e uma indicação oportuna de quão completamente o Claude da Anthropic se infiltrou na comunidade de desenvolvedores e o crescente interesse em tokenmaxxing. Essa nova tendência de "produtividade" vê engenheiros de software em várias empresas de tecnologia maximizando o número de tokens de IA consumidos no trabalho como uma medida de quanto eles abraçou a IA.

Como um usuário do Reddit brincou ao ver o projeto pela primeira vez: "Neste momento, o Anthropic deve apenas enviá-los para nós de graça."

Outro sugeriu adicionar um botão para aumentar a capacidade ou recarregar mais tokens usando seu cartão arquivado. (Ha, isso pode ser perigoso!)

A ideia do projeto vem de Reykjavik, o desenvolvedor de software baseado na Islândia Hermann Haraldsson , que diz que sempre quis brincar com dispositivos embarcados, mas nunca teve tempo.

"Não sou um desenvolvedor incorporado ou algo assim", disse Haraldsson ao TechCrunch em uma ligação. Mas Claude conseguiu guiá-lo pelo projeto em apenas alguns dias, disse ele. “É realmente um acesso democratizado à programação, para que qualquer pessoa possa agora fazer o que os desenvolvedores costumavam fazer. Acho que isso é muito positivo, na verdade.”

A maior parte do tempo que ele passou construindo o dispositivo foi focado no design, certificando-se de obter a fonte, as cores e as pequenas animações certas.
