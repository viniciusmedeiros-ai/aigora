A Anthropic expandiu o Claude Code com suporte a subagentes especializados, melhor integração com VS Code e JetBrains, e novos hooks para automação de fluxos de trabalho complexos.

As novas funcionalidades permitem que times de desenvolvimento criem pipelines completos de código, teste e deploy sem sair do terminal.
