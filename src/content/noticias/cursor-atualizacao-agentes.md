O editor de código com IA Cursor lançou um modo agente que executa tarefas de desenvolvimento de forma autônoma, desde a análise de bugs até a criação de features completas com testes.

A atualização posiciona o Cursor como concorrente direto do GitHub Copilot Workspace e do próprio Claude Code para fluxos de desenvolvimento autônomo.
