Sessão de áudio OpenAI WebRTC, agora com contexto de documento. Construí a primeira versão desta ferramenta em dezembro de 2024 para experimentar a então nova API OpenAI WebRTC para interagir com seus modelos de áudio em tempo real.

No mês passado, a OpenAI apresentou um novo modelo para essa API chamado GPT‑Realtime‑2 , que eles promoveram como "nosso primeiro modelo de voz com raciocínio GPT‑5‑class" - com um corte de conhecimento em 30 de setembro de 2024.

Estive esperando que esse modelo aparecesse no aplicativo ChatGPT para iPhone, mas ainda não apareceu, então revisitei meu antigo playground.

Agora você pode escolher o melhor modelo e também pode colar uma grande parte do contexto do documento para que possa ter uma conversa de áudio em seu navegador sobre qualquer informação que você ache útil explorar de maneira conversacional.
